html5 超文本标记语言
< title> 标签对搜索引擎优化 (SEO) 非常重要，因为它决定了网页在搜索结果中的标题。
< meta> 标签可以提供额外的信息，如描述和关键词，帮助搜索引擎更好地理解网页内容。 < link> 标签用于链接外部资源，如样式表或图标。
< script> 标签用于嵌入或链接 JavaScript 代码，增强网页的交互性。
< style> 标签用于嵌入 CSS 样式，控制网页的外观和布局。
< body> 标签包含网页的主要内容，是用户在浏览器中看到的部分。
< header> 标签通常包含网站的导航和标题信息。
< footer> 标签通常包含版权信息、联系信息或其他辅助内容。
< nav> 标签用于定义导航链接，帮助用户在网站中快速找到所需内容。
< article> 标签用于定义独立的内容块，如博客文章或新闻报道。
< section> 标签用于定义文档中的章节或部分，帮助组织内容。
< aside> 标签用于定义与主内容相关的侧边栏或附加信息。
< h1> 到 < h6> 标签用于定义标题，< h1> 是最高级别的标题，< h6> 是最低级别的标题。
< p> 标签用于定义段落，是文本内容的基本单位。
< a> 标签用于创建超链接，可以链接到其他网页或资源。
< img> 标签用于嵌入图像，支持多种格式，如 JPEG、PNG 和 GIF。
< video> 标签用于嵌入视频内容，支持多种格式，如 MP4、WebM 和 Ogg。
< audio> 标签用于嵌入音频内容，支持多种格式，如 MP3、WAV 和 Ogg。
< form> 标签用于创建交互式表单，允许用户输入数据并提交。
< input> 标签用于创建各种类型的输入字段，如文本框、复选框和单选按钮。
< button> 标签用于创建可点击的按钮，触发特定操作或提交表单。
< select> 标签用于创建下拉列表，允许用户从多个选项中选择。
< textarea> 标签用于创建多行文本输入区域，适合长文本输入。
< table> 标签用于创建表格，组织和展示数据。
< tr> 标签用于定义表格行，< th> 标签用于定义表头单元格，< td> 标签用于定义表格数据单元格。
< ul> 标签用于创建无序列表，< ol> 标签用于创建有序列表，< li> 标签用于定义列表项。
< div> 标签用于创建通用容器，帮助组织和布局网页内容。
< span> 标签用于定义行内元素，通常用于样式或脚本操作。
< blockquote> 标签用于引用其他来源的内容，通常用于引述或引用文本。
< code> 标签用于定义代码片段，通常用于显示编程代码或命令。
< pre> 标签用于定义预格式化文本，保留空格和换行符，通常用于显示代码或诗歌。
< strong> 标签用于强调文本，通常以粗体显示。
< em> 标签用于强调文本，通常以斜体显示。
< small> 标签用于定义小号文本，通常用于版权信息或注释。
< del> 标签用于定义已删除的文本，通常以删除线显示。
< ins> 标签用于定义已插入的文本，通常以下划线显示。
< mark> 标签用于定义高亮文本，通常以黄色背景显示。
< sub> 标签用于定义下标文本，通常用于化学式或数学公式。
< sup> 标签用于定义上标文本，通常用于数学公式或引用。
< time> 标签用于定义时间或日期，通常用于标记事件的时间。
< progress> 标签用于显示任务的进度，通常用于表示下载或上传进度。
< meter> 标签用于显示测量值，如温度或电池电量，通常用于表示范围内的值。
< details> 标签用于创建可展开的内容块，通常用于显示额外信息或选项。
< summary> 标签用于定义 < details> 标签的摘要或标题，通常用于显示可展开内容的标题。
< dialog> 标签用于创建对话框或模态窗口，通常用于显示提示或确认信息。
< template> 标签用于定义可重用的 HTML 模板，通常用于动态内容生成。
< canvas> 标签用于创建可绘制的画布，通常用于图形和动画。
< svg> 标签用于嵌入可缩放矢量图形，通常用于图标和图形。
